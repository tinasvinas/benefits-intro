📋 第 1 步 · 智能审题
你的目标是定义一个 int 变量 x，然后用指针访问它的值并打印。

🔍 第 2 步 · 代码分析
代码第 3 行：char *p = &x;
这里 &x 的类型是 int*（指向 int 的指针），但你声明的 p 是 char*（指向 char 的指针）。类型不一致，所以编译器报错。

💡 第 3 步 · 关键点拨
在 C 语言中，指针不只记录"地址"，还记录"该地址处数据的类型"。
int* 知道每次跳 4 字节（在 32/64 位系统中 int 通常是 4 字节）
char* 知道每次跳 1 字节
这两种指针在内存解读规则上完全不同，不能随便互转。
